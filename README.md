### Задание (массивы)
1. Вывести квадратную матрицу по заданному N.\n

Пример N = 3:\n
Пример:\n
1 2 3\n
0 1 1\n
4 5 6\n

2. Вывести заданный массив размером N в обратном порядке.\n

Ввод-> 1 2 3 4 5\n
Вывод -> 5 4 3 2 1.


3. Заполнить верхний треугольник матрицы 1, а нижний 0.\n

Пример N = 3:\n
Пример:\n
0 0 1\n
0 1 1\n
1 1 1\n


4. Заполнить матрицу числами от 1 до N^2 улиткой.

Пример N = 5:\n
Пример:\n
1  2  3  4  5\n
16 17 18 19 6\n
15 24 25 20 7\n
14 23 22 21 8\n
13 12 11 10 9\n
